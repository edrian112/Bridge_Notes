# Bridge Notes - Phase 1 테스트 가이드

## 🚀 Chrome에 확장 프로그램 로드하기

### 1단계: Chrome 확장 프로그램 페이지 열기

```
chrome://extensions
```

또는:
- Chrome 메뉴 → 도구 더보기 → 확장 프로그램

### 2단계: 개발자 모드 활성화

- 우측 상단의 **"개발자 모드"** 토글을 ON으로 설정

### 3단계: 확장 프로그램 로드

1. **"압축해제된 확장 프로그램을 로드합니다"** 버튼 클릭
2. `bridge-notes` 폴더 선택
3. "선택" 버튼 클릭

✅ Bridge Notes가 확장 프로그램 목록에 나타나면 성공!

## 🧪 Phase 1 기능 테스트

### 테스트 환경 준비

1. **Claude.ai 접속**: https://claude.ai
2. Claude와 간단한 대화 진행 (3-4번 왕복)

### 테스트 시나리오

#### ✅ Test 1: 확장 프로그램 활성화

1. Chrome 툴바에서 Bridge Notes 아이콘 클릭
2. **예상 결과**:
   - 팝업이 열림
   - "범위 선택 시작" 버튼 표시
   - "최근 캡처 (0)" 섹션 표시

#### ✅ Test 2: 범위 선택 모드

1. "범위 선택 시작" 버튼 클릭
2. **예상 결과**:
   - 팝업이 자동으로 닫힘
   - 페이지 상단에 "🎯 시작 지점을 클릭하세요" 오버레이 표시
   - ESC 키로 취소 가능

#### ✅ Test 3: 시작 지점 선택

1. Claude의 첫 번째 답변 메시지 클릭
2. **예상 결과**:
   - 선택한 메시지에 녹색 테두리 표시
   - 오버레이 메시지 변경: "✅ 끝 지점을 클릭하세요"

#### ✅ Test 4: 끝 지점 선택 및 미리보기

1. 마지막 답변 메시지 클릭
2. **예상 결과**:
   - 선택한 메시지에 주황색 테두리 표시
   - 미리보기 모달이 중앙에 표시됨
   - 캡처된 텍스트 미리보기 (최대 500자)
   - "총 N자 캡처됨" 통계 표시

#### ✅ Test 5: 개인정보 감지 (선택적)

대화에 이메일이나 전화번호를 포함시킨 경우:
- **예상 결과**:
  - 미리보기 상단에 노란색 경고 배너 표시
  - "⚠️ 이메일, 전화번호 등 개인정보가 감지되었습니다" 메시지

#### ✅ Test 6: 클립보드 복사

1. "📋 클립보드에 복사" 버튼 클릭
2. **예상 결과**:
   - 버튼 텍스트가 "✅ 복사 완료!"로 변경
   - 버튼 색상이 초록색으로 변경
   - 1.5초 후 모달 자동 닫힘
3. 텍스트 에디터에 붙여넣기 (Cmd+V)
4. **예상 결과**:
   - 대화 형식으로 텍스트가 붙여넣어짐
   - 예시:
     ```
     사용자: 질문 내용

     Claude: 답변 내용

     사용자: 추가 질문

     Claude: 추가 답변
     ```

#### ✅ Test 7: 최근 캡처 목록

1. Bridge Notes 아이콘 다시 클릭
2. **예상 결과**:
   - "최근 캡처 (1)" 표시
   - 방금 캡처한 내용의 미리보기 표시
   - "방금 전" 시간 표시
3. 캡처 항목 클릭
4. **예상 결과**:
   - 클립보드에 자동 복사
   - "클립보드에 복사되었습니다!" 메시지

#### ✅ Test 8: 다시 선택

1. 캡처 후 미리보기 모달에서 "🔄 다시 선택" 버튼 클릭
2. **예상 결과**:
   - 모달이 닫힘
   - 선택 모드가 다시 활성화됨
   - 새로운 범위 선택 가능

#### ✅ Test 9: ESC 키로 취소

1. 선택 모드 활성화
2. ESC 키 누르기
3. **예상 결과**:
   - 선택 모드 비활성화
   - 오버레이 사라짐
   - 모든 하이라이트 제거

#### ✅ Test 10: 지원되지 않는 사이트

1. Google.com 등 다른 사이트 접속
2. Bridge Notes 아이콘 클릭
3. **예상 결과**:
   - 빨간색 에러 메시지 표시
   - "이 사이트는 지원되지 않습니다. Claude.ai, ChatGPT, Perplexity에서 사용해주세요."
   - "범위 선택 시작" 버튼 비활성화 (회색)

## 🐛 예상 가능한 문제 및 해결

### 문제 1: 확장 프로그램이 로드되지 않음

**원인**: manifest.json 오류
**해결**:
```bash
# Chrome 개발자 도구에서 오류 확인
chrome://extensions
# "오류" 버튼 클릭하여 상세 내용 확인
```

### 문제 2: Content script가 주입되지 않음

**증상**: 버튼 클릭 시 "페이지와 통신할 수 없습니다" 에러
**해결**:
1. Claude.ai 페이지 새로고침 (F5)
2. 확장 프로그램 재로드
3. 다시 시도

### 문제 3: 아이콘이 보이지 않음

**원인**: 아이콘 파일 경로 오류
**확인**:
```bash
ls -la /Users/admin/Documents/dev/bridge-notes/icons/
# icon16.png, icon48.png, icon128.png 존재 확인
```

### 문제 4: 텍스트 추출이 이상함

**원인**: DOM 구조 변경 또는 선택자 불일치
**임시 해결**:
- Chrome DevTools (F12) 열기
- Console 탭에서 에러 메시지 확인
- 문제 보고

### 문제 5: 클립보드 복사 실패

**원인**: 브라우저 권한 문제
**해결**:
1. Chrome 설정 → 개인정보 및 보안 → 사이트 설정
2. 클립보드 권한 확인
3. claude.ai 허용 확인
